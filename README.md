## 💻 课程作业

在这里你可以找到我的编程作业解决方案。每一次作业都是向着构建和理解大型语言模型复杂组件迈出的一步。


| 作业编号 | 主题 | 描述 | 状态 | 难度 |
|:---:|:---:|:---:|:---:|:---:|
| **HW1** | 基础知识(Transformer 架构详解) | 深入探讨语言模型的基本构建模块，包括 Tokenizer、Embedding 等 | ✅ 已完成 | ⭐⭐⭐⭐⭐ |
| **HW2** | 并行优化与Triton实现 | 探索训练大型模型的优化方法，包括 Flash Attention、数据并行等 | ✅ 已完成 | ⭐⭐⭐⭐⭐ |
| **HW3** | Scaling Law | 了解大模型的第一性原理，研究模型规模与性能的关系 | ✅ 已完成 | ⭐ |
| **HW4** | 完整数据清洗流程 | 实现对网页数据的完整的数据清洗与质量分类流程 | 待更新 | ⭐⭐⭐ |
| **HW5** | LLM + RL | 实现前沿的GRPO算法等 | 待更新 | ⭐⭐⭐⭐⭐ |
